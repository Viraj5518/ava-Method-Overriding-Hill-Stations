# Java Method Overriding - Hill Stations

## Overview

This project demonstrates the concept of **Method Overriding** in Java using different hill station classes. A parent class defines a common method, while child classes override it to provide their own implementation, showcasing runtime polymorphism.

---

## Features

- Demonstrates Method Overriding in Java.
- Uses inheritance to create specialized hill station classes.
- Displays unique information for different hill stations.
- Shows Runtime Polymorphism through overridden methods.
- Simple and easy-to-understand object-oriented design.

## How to Run

1. Clone or download this repository.
2. Open the project in Eclipse or any Java IDE.
3. Compile all Java files.
4. Run the `Main.java` file.
5. Observe how each hill station overrides the parent class method.
